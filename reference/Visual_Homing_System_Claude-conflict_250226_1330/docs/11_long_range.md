# Розширення радіусу дії Visual Homing

## Проблема

WiFi на Raspberry Pi має обмежений радіус дії (~50-100м на відкритій місцевості).
Для реальних польотів потрібен контроль на відстанях 1-10+ км.

## Рішення

## Варіант 1: SiK Radio (рекомендовано для початку)

### Обладнання
- **SiK Radio 433MHz** (Україна/Європа) або **915MHz** (США)
- Два модулі: один на дрон, один на землю
- Ціна: ~$30-50 за пару

### Характеристики
- Дальність: 1-5 км
- Швидкість: 57600 baud
- Напівдуплекс

### Підключення на Pi
```
Pi Zero 2W          SiK Radio
---------           ---------
GPIO14 (TX) ───────► RX
GPIO15 (RX) ◄─────── TX
5V ─────────────────► VCC
GND ────────────────► GND
```

### Конфігурація
```python
# /home/pi/visual_homing/config.py
MAVLINK_PORT = "/dev/serial0"  # для SiK Radio
MAVLINK_BAUD = 57600
```

### На землі
```bash
# Linux/Mac
screen /dev/ttyUSB0 57600

# Windows - використовуйте Mission Planner
# COM порт → 57600 baud
```

## Варіант 2: RFD900x (професійний)

### Характеристики
- Дальність: 5-40+ км
- Потужність: до 1W
- Ціна: ~$200-400 за пару

### Переваги
- Вбудований PPM/SBUS для RC
- Подвійна антена (diversity)
- Mesh networking можливість

---

## Варіант 3: 4G/LTE моду|ем (необмежений радіус)

### Обладнання
- **Huawei E3372** USB модем (~$30)
- SIM-картка з інтернетом

### Архітектура
```
Дрон (Pi + 4G) ──► Інтернет ──► VPN сервер ◄── Ноутбук/Телефон
```

### Налаштування на Pi
```bash
# Встановити NetworkManager
sudo apt install network-manager modem-manager

# Налаштувати з'єднання
sudo nmcli c add type gsm ifname "*" con-name "4g" apn "internet"
sudo nmcli c up "4g"
```

### Переваги
- Необмежений радіус (де є покриття)
- Двосторонній зв'язок
- Можна передавати відео

### Недоліки
- Затримка 50-200мс
- Потрібне покриття мережі
- Абонплата

## Варіант 4: Аналогове FPV відео

### Обладнання
- **TX**: Відеопередавач 5.8GHz, 25-600mW
- **RX**: FPV приймач + очки або монітор

### Підключення камери
```
Pi Camera/Thermal ──► Pi ──► AV Out ──► VTX ──►))) 5.8GHz
                                                    │
                                              ◄───)))) VRX ──► FPV очки
```

### Переваги
- Затримка <50мс
- Дальність 1-10км
- Працює завжди (не потребує IP)

## Інтеграція з Visual Homing

### 1. Автоматичне перемикання

```python
# В main.py додати логіку:
class CommunicationManager:
    def __init__(self):
        self.wifi_connected = False
        self.radio_connected = False
        self.lte_connected = False
    
    def get_best_link(self):
        """Вибрати найкращий канал зв'язку"""
        if self.wifi_connected:
            return "wifi"  # Низька затримка
        elif self.radio_connected:
            return "radio"  # Надійний
        elif self.lte_connected:
            return "lte"  # Резервний
        return None
```

## Висновок

Для вашого випадку рекомендую почати з:

1. **SiK Radio 433MHz** — для телеметрії та керування (2-5 км)
2. **Аналогове FPV** — для відео в реальному часі (2-5 км)
3. **WiFi** — для налаштування та близької роботи

Це дасть надійний контроль на відстані до 5 км за ~$80-100 додатково.
